# Proyecto Firma Electrónica Avanzada

Sistema desarrollado en JavaScript y Node.js para simular una firma electrónica avanzada.

## Funciones
- Generación de hash SHA-256
- Firma digital con mouse
- Verificación de integridad
- Descarga de documentos firmados
- Registro de documentos firmados

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Node.js
- Express
